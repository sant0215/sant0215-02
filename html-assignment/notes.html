Notes

Here is a list of a few tags I've learned so far:

img
Stands for image. Used to display images on webpages.
a
Stands for anchor. Used to link to other pages, websites or other parts of the webpage.
table
The main container tag to create tables.
thead
Tag used to add a heading row for table columms.
ol
Stands for ordered list. This is used for lists where the order of items is relevant.
