Bootstrap in a nutshell

Open source.

Learn everything you need to know to get started building your responsive site with all the documentation readily available for anyone to use.

Components ready to use.

You can create your website through Bootstrap's extensive components library including different elements to enhance your website.

No JavaScript? No Problem.

You can use Bootstrap components even without any prior knowledge to JavaScript. This makes simple functions normally needing JavaScript more accessible.

Ready to go whenever.

Getting started has never been so fast. Just use the starter template provided in the documentation and you're good to go.

Why use Bootstrap?

  • It's fast.

    Libraries of components are already made for you. You can apply these on any of your designs with complete flexibility without even touching CSS.

  • It's flexible.

    Bootstrap supports all latest and stable releases of all major browsers and platforms. Not only that, components easily flow well with each other without the need of extra code.

  • It's free.

    All of this is completely accessible through the Bootstrap website, and is continuously being developed.
